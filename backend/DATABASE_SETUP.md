# Database Setup Instructions

## Prerequisites
- PostgreSQL installed and running
- Python 3.7+

## Quick Start

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Start PostgreSQL** (if using Postgres.app, just open it)

3. **Create the database:**
   ```bash
   createdb laundry_db
   ```

4. **Run the setup:**
   ```bash
   python database.py
   ```
